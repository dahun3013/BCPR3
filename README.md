# BCPR3 3차 프로젝트
급격하게 진행된 수업 플랫폼의 전환으로 인한, 학교와 비대면 플랫폼 간에 다양한 문제들 중, <br>
비대면 수업의 문서화에 대한 어려움을 해결할 수 있는 문서 자동화 보조 웹사이트를 구축한다. <br><br>
